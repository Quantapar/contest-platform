# Cypher

A fullstack contest platform where creators can host MCQ and DSA contests, and users can participate, submit solutions, and compete on a leaderboard

This repository contains **both frontend and backend** code.

---

## Features

- User authentication (JWT)
- Role-based access (creator / contestant)
- Create and manage contests
- MCQ & DSA problem support
- Submit solutions
- Live leaderboard
- Clean, modular backend + frontend separation

---

## Tech Stack

### Frontend

- React
- TypeScript
- Tailwind CSS

### Backend

- Bun
- TypeScript
- Express
- Prisma ORM
- PostgreSQL
- JWT Authentication
- Zod validation
